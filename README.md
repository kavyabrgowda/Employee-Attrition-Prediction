# Employee Retention Predictor (HR Analytics)

---

## Project Overview
This project predicts which employees might leave the company.  
It helps HR managers identify **high-risk employees** and take action to retain them.  

---

## Tools Used
- Python, Pandas, Scikit-learn, Seaborn  
- Flask (for web dashboard)

---

## Features
- Analyze employee data (age, department, job role, salary, satisfaction, tenure)  
- Predict attrition for single or multiple employees  
- Show **risk level**: High, Medium, Low  
- Identify **key factors** causing attrition  
- Provide **HR recommendations** to retain employees  

---

## Outcome
The system product into a smart tool that helps reduce employee turnover and improve retention using AI predictions.
